# VertexFlow

### Visualize Graph Algorithms Like Never Before

VertexFlow is a modern interactive graph algorithm visualization platform built to help students, developers, educators, and interview candidates understand graph algorithms through real-time visual learning.

Live Demo: https://vertexflow-sigma.vercel.app/

---

## Preview

VertexFlow allows users to:

* Build graphs manually
* Create weighted and directed graphs
* Visualize graph algorithms step-by-step
* Compare algorithms side-by-side
* Learn graph concepts interactively
* Analyze traversal behavior in real time

---

## Features

### Interactive Graph Playground

Create and edit graphs visually.

* Add nodes manually
* Connect nodes with edges
* Directed / Undirected graphs
* Weighted edges
* Drag and reposition nodes
* Delete nodes and edges
* Graph statistics panel
* Import / Export support

### Algorithm Visualization

Watch algorithms execute step-by-step.

Supported features:

* Live traversal animations
* Timeline controls
* Step-by-step execution
* Story mode explanations
* Internal data structure visualization
* Complexity analysis

### Comparison Mode

Compare algorithms on the same graph.

Compare:

* Runtime
* Steps executed
* Nodes visited
* Memory usage
* Traversal order

Side-by-side visual execution makes performance differences easy to understand.

### Learning Hub

Comprehensive educational content for every algorithm.

Includes:

* Theory
* Complexity analysis
* Pseudocode
* Use cases
* Interview tips
* Common mistakes
* Real-world applications

# Supported Algorithms

## Traversal

* Breadth First Search (BFS)
* Depth First Search (DFS)
* Bidirectional BFS

## Shortest Path

* Dijkstra Algorithm
* Bellman-Ford Algorithm
* Floyd-Warshall Algorithm
* A* Search

## Minimum Spanning Tree

* Prim's Algorithm
* Kruskal's Algorithm

## Directed Graph Algorithms

* Topological Sort
* Kosaraju Algorithm
* Tarjan Algorithm

## Connectivity

* Connected Components
* Disjoint Set Union (DSU)

## Cycle Detection

* Directed Cycle Detection
* Undirected Cycle Detection

# Installation

Clone the repository:

```bash
git clone https://github.com/karalapatiphanicharan-cyber/vertexflow.git
```

Move into the project:

```bash
cd vertexflow
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
